# Claude AI Project Instructions

This project is a personal portfolio built with Next.js.

## Goal
Create a modern, fast, and responsive developer portfolio.

## Design Style
- Minimal but attractive UI
- Smooth animations (optional)
- Dark/light friendly design

## Behavior Rules
- Always generate clean and optimized code
- Avoid bloated components
- Focus on performance and readability

## Pages
- Home: Intro + hero section
- About: Personal info + skills
- Services: What I offer
- Contact: Simple form or social links

## Important
- Keep everything production-ready
- Avoid experimental or unstable patterns
